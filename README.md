# Venturus-1.0.0

## Prerequisites
- npm
````bash
npm install npm@latest -g 
````
## Installation

1. Get a free API Key at https://www.api-football.com/

2. Clone the repo
````bash
git clone https://github.com/Kai-Ribeiro/venturus.git
````

3. Install NPM packges
````bash
npm install
````
4. Enter your API in .env
````bash
 REACT_APP_KEY = 'ENTER YOUR API'
````
